对象相等比较

Aug 21, 2019Scala

在 Scala 中，可以直接使用JDK的类库，好多功能语法相似，在Java中比较对象就用==与equals很简单。在Scala中可以使用==、equals、eq、sameElements

比较规则

  • Java中比较引用或则基本数据类型用==，比较内容用equals方法
  • Scala中直接使用==，内部首先判断左侧是否为null，如果不为null，直接调用equals方法，当然也可以调用equals方法
  • Scala中如果要特别比较引用相等性可以用eq与ne方法，这个跟Java中的==一直

https://www.scala-lang.org/api/current/scala/AnyRef.html

final def==(arg0: Any): Boolean
The expression x == that is equivalent to if (x eq null) that eq null else x.equals(that).
如果x为null就that.eq(null)，如果x不为null就x.equals(that)
final def eq(arg0: AnyRef): Boolean
Tests whether the argument (that) is a reference to the receiver object (this).
def equals(arg0: AnyRef): Boolean
The equality method for reference types.
